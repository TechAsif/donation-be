## Installation
Node Version: 14.17.3
```bash
$ npm install -g @nestjs/cli@10.0.0
$ npm install -g ioredis@5.4.1
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
